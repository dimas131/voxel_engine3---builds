# Controls
1. WASD - move
2. SPACE - move up
3. LEFT SHIFT - move down
4. LMB - break bloks
5. RMB - putting blocks
6. ESC - close game
7. TAB - toogle cursor
8. F1 - save game
9. F2 - load game
10. NUMBERS 1-4 - choose bloks
11. NUMPAD ADD(+) - increase speed
12. NUMPAD SUBTRACT(-) - decrease speed
13. LEFT CONTROL(CTRL) - speed up
